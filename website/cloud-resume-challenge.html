Cloud Resume Challenge

This very website is a testament to my cloud engineering capabilities. Built as a part of the Cloud Resume Challenge, it's a serverless platform continuously deployed and updated, showcasing practical experience with a modern cloud stack.

Core Technologies Currently in Use:

  • AWS:
    • S3 (Static Hosting)
    • CloudFront (CDN)
    • Route 53 (DNS)
  • GitHub Actions: For seamless, automated CI/CD and deployment, ensuring this site is always up-to-date.

Future Enhancements & Learning:

I'm continuously expanding this project, with plans to integrate additional AWS services to enhance its functionality, including:

  • AWS Lambda: To power a backend API for dynamic features.
  • AWS API Gateway: For secure exposure and management of the API.
  • AWS DynamoDB: To serve as a NoSQL database, specifically for tracking visitor count and other potential data.
